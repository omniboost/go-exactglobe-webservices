Additional remarks for MailMergePending

This entity mapped to MailMergePending repository business component and provides CRUD fiunctionality fot MailMergePending table.

if Exact Word Merge is installed in Synergy folder, after creating new record using this entity a notification will be sent to Exact.Notification.Subscribers.WordMerge, and that class creates a message in MSMQ for Word Merge resolver to pick it up for processing.
